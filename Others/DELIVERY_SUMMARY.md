## ⚡ QUICK START (Do This Now!)

### Step 1: Configure Contact Form (5 minutes)
```
1. Open https://formspree.io/
2. Create account → Create new form
3. Copy your form ID (e.g., f_abc123xyz)
4. Open contact.html
5. Find: <form id="contactForm" action="https://formspree.io/f/xyzabnop"
6. Replace "xyzabnop" with your ID
7. Save and test!
```

### Step 2: Update Business Info (10 minutes)
```
Search & Replace in ALL HTML files:
- "Siva Medicals" → Your company
- "9952930484" → Your phone
- "Madambakkam" → Your location
- "8 AM - 10 PM" → Your hours
```

### Step 3: Test Website (5 minutes)
```
1. Open index.html in browser
2. Click "Mode" button → Dark mode works ✓
3. Scroll page → Animations trigger ✓
4. Go to Contact → Fill form & submit ✓
5. Resize to mobile → Responsive ✓
```

```

2. Drag & drop your folder
3. Done! Get live URL instantly

ALTERNATIVE: Traditional hosting
1. Upload via FTP
2. Set DNS records
3. Done!
```
